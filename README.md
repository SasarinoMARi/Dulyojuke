# 들려줄게

Youtube에서 음원을 다운받고 태그 작업까지 진행합니다.
태그는 현재 제목, 아티스트, 앨범명, 앨범아트만 지정 가능합니다.

### [Download](https://github.com/Usagination/Dulyojuke/releases/)
## Usage

보면 느낌이 올겁니다.
